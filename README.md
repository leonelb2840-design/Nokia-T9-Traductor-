# 📱 Nokia T9 Master - LeonelB_Store

Un potente traductor web de código **Nokia T9** (multitap) diseñado para creadores de contenido, fans de lo retro y desarrolladores. Convierte texto a secuencias numéricas y viceversa con un solo clic.

## ✨ Características
- **Conversión Dual:** Traduce letras a números y códigos numéricos a texto.
- **Optimizado para Móviles:** Botones de copiado reforzados y pegado manual/automático.
- **Interfaz Retro:** Estilo basado en terminales clásicas con logo ASCII.
- **Limpieza Rápida:** Botón dedicado para resetear todos los campos.

## 🚀 Cómo usar
1. **Letras a Números:** Escribe tu mensaje en el cuadro superior. El código aparecerá instantáneamente.
2. **Números a Letras:** Pega o escribe los números (separados por espacios) en el cuadro inferior para descifrar el mensaje.
3. **Copiar:** Usa los botones verdes para guardar el resultado en tu portapapeles.

## 🛠️ Tecnologías
- **HTML5**
- **CSS3** (Diseño Dark Mode & Responsive)
- **JavaScript ES6** (Lógica de mapeo y API de portapapeles)
